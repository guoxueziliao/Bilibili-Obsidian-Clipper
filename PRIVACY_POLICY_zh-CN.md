# Bilibili Obsidian Clipper 隐私政策

最后更新：2026-04-09

## 我们收集什么

本扩展不会将你的数据上传到开发者自有服务器。  
扩展仅在本地处理以下信息：

- 当前 Bilibili 视频页 URL（用于识别视频与分 P）
- 字幕内容（用于预览、复制、下载、写入 Obsidian）
- 用户设置（如导出格式、笔记目录、Local REST API 地址、API Key）

## 我们如何使用数据

数据仅用于扩展核心功能：

- 抓取并展示当前视频字幕
- 导出字幕（复制到剪贴板或下载文件）
- 按你的设置写入本地 Obsidian（通过 Local REST API）

## 数据存储与传输

- 设置数据存储在浏览器本地（`chrome.storage.local`）
- 不会出售用户数据
- 不用于广告或画像
- 不用于与扩展核心功能无关的用途
- 不向开发者自有远端服务器传输用户数据

## 第三方连接说明

扩展会访问以下服务以完成功能：

- `bilibili.com` / `api.bilibili.com` / `*.hdslb.com`：读取视频与字幕数据
- `127.0.0.1` 或 `localhost`：连接你本机的 Obsidian Local REST API（如果你启用该功能）

## 你的控制权

- 你可以随时在扩展设置中修改或清空配置
- 你可以禁用或卸载扩展来停止数据处理
